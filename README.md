# UE3DGame

[프롭나이트], [데드바이데이라이트] 게임을 모티브로 한
PC기반 3D 멀티플레이 대전 게임

생존자 진영 플레이어들은 사물 변신 기능을 가지며, 각 지점의 발전기를 돌려 탈출하는 것이 목표이고,   
킬러 진영 플레이어는 홀로 생존자들을 불능 상태로 만드는 것이 목표입니다. 

기본 조작은 wasd, 스페이스 점프, 마우스 좌,우 클릭 등
